# Life OS-pro 🚀

مشروع متكامل لإدارة العادات والمهام اليومية باستخدام React و Python.

## 📁 هيكل المشروع (Project Structure)
تم تقسيم المشروع لضمان أعلى درجة من التنظيم:

* **`frontend/`**: يحتوي على واجهة المستخدم (React JS).
* **`backend/`**: خاص بالعمليات الذكية وقاعدة البيانات (FastAPI & Firebase).
* **`docs/`**: ملفات التخطيط والـ UI/UX والوثائق الرسمية.
* **`scripts/`**: أدوات مساعدة للأتمتة والتهيئة.

## 🛠 بيئة العمل الحالية
* **Main Branch**: النسخة المستقرة.
* **Dev Branches**: يتم العمل حالياً على `dev-frontend` لتأسيس الواجهة.
